Что это?

Этот сайт помогает почистить ваш список друзей друзей "VK".
Он выводит друзей и дату вашего последнего общения с ними.

Исходный код приложения доступен на Github, Вы можете сами удостовериться, что пользоваться сайтом безопасно.

Как авторизоваться?

Сайт использует API ВКонтакте, чтобы загрузить список ваших друзей и диалоги. Поэтому для его использования требуется войти "ВКонтакте".

  1. Перейдите по ссылке "Войти VK" на главной странице сайта.
  2. Откроется окно авторизации, затем будут запрошены разрешения приложения для доступа к сообщениям и друзьям.
  3. Затем вы будете переадресованы на страницу "https://oauth.vk.com/blank.html".
  4. Скопируйте ссылку из адресной строки в поле ввода на главной странице сайта и нажмите "Поехали".

Это безопасно?

К сожалению API ВКонтакте не позволяет веб-сайтам другим способом получить доступ к сообщениям пользователя. Обычно это доступно только мобильным и desktop приложениям. Текущий способ авторизации - обход этого ограничения.
На странице "https://oauth.vk.com/blank.html" написано не копировать ссылку на сторонние сайты. Это предупреждение существует потому, что после авторизации приложение может получить контроль над данными вашей страницы.
Но VK Cleaner безопасен.
Данный сайт делает с полученным доступом только три вещи:

  • Получает ваше имя, фамилию и фотографию.
  • Получает списки ваших друзей и диалогов.
  • Позволяет удалять друзей.
Ваши никуда не пересылаются. Список ваших диалогов и друзей хранится в локальной памяти не более шести минут. После нажатия кнопки "Выйти" все следы вашего присутствия на сайте стираются.
